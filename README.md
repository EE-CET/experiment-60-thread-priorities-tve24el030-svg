# Experiment 60: Thread Priorities

## Problem Statement

Write a program to create three threads.
* Set the priority of the first thread to `MIN_PRIORITY`.
* Set the priority of the second thread to `NORM_PRIORITY`.
* Set the priority of the third thread to `MAX_PRIORITY`.

Display the name and priority of each thread.

## Input Format

* No Input.

## Output Format

* `Thread 1 Priority: 1`
* `Thread 2 Priority: 5`
* `Thread 3 Priority: 10`

### Example 1

**Input:**

```text
No input
```

**Output:**

```text
Thread 1 Priority: 1
Thread 2 Priority: 5
Thread 3 Priority: 10
```
